📌 Step 2: Feature List for Phase 1

These are the expected features that the app will include in the first phase build.
🛠️ Core Features

    Super smooth animations & UI responsiveness
    Adaptive design for different screen sizes
    Secure authentication (email & social login)
    End-to-end encrypted user sessions
    Optimized media storage for images & videos
    Gesture-based navigation

📅 Memory Timeline (Core Feature)

    Chronological ordering of posts
    Create posts with text, images & videos
    Edit old posts (no delete feature)
    Pinned milestones
    Categorization (Personal, Career, Travel, etc.)
    Search & filter past memories

💬 Social Features

    Follow/unfollow users
    Like & comment on posts
    Tag people in memories
    Basic notifications for interactions

🛠️ User Profiles & Settings

    View & edit profile
    Profile bio, image & other details
    Privacy controls (public/private posts)

📌 Step 3: Feature Mapping (Which Screen Has What Features?)
Screen	Features Included
Onboarding Screen	App introduction, login/signup redirection
Login & Signup Screen	Email & social login, authentication checks
Timeline (Main Feed)	Chronological posts, smooth scrolling, memory categories, user interactions
Post Details Screen	Full post view, like/comment, edit option (for own posts)
Create New Memory Screen	Add text, images, videos, tags, and privacy options
Edit Memory Screen	Modify existing posts without deleting
Explore Page	Trending posts, suggested people, popular memories
Search Screen	Search by date, tags, categories, or keywords
My Profile Page	User bio, profile picture, all posted memories
Other User’s Profile	View another user’s public memories, follow/unfollow option
App Settings	Dark mode, theme adjustments, notifications toggle
Notification Center	List of likes, comments, tags, and follows
Privacy Settings	Control memory visibility, blocked users
