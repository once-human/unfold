# Project Unfold (Codename)

A cross-platform social media application that builds and maintains a chronological timeline of your life story.

## Project Overview

This is a development project with the codename "Unfold". The app offers a unique approach to social media by organizing all shared content chronologically into a personal life timeline, rather than as disconnected posts.

## Core Concept

Unlike traditional social media platforms, this application:

- Automatically organizes posts in chronological order
- Creates a navigable timeline of your life experiences
- Allows editing and refining of past content instead of deletion
- Connects memories with people and places

## Key Features

### Data Structure
- Date & Time-based organization of posts
- Life section categories (Personal, Career, Travel, etc.)
- Memory connections with people and places

### User Experience
- Vertical timeline scrolling interface
- Pinned milestones for major life events
- Edit-don't-delete approach to content management
- Auto-generated yearly recaps

### Social Features
- Memory linking between friends
- Monthly life reports
- Throwback prompts for revisiting past memories

## Technical Implementation

The application is built using Flutter for cross-platform compatibility:
- Web application
- Android application
- iOS application
- Windows desktop
- macOS desktop
- Linux desktop

Initial development focus is on the web and Android platforms.



## Development Status

Currently in early development stage. This is an internal codename project.

## Getting Started

### Prerequisites
- Flutter (latest stable version)
- Dart SDK
- Platform-specific development tools for your target platforms

### Installation

1. Clone the repository
2. Install dependencies:
   

flutter pub get


3. Run the application:
   

# For Web
   flutter run -d chrome
   
   # For Android
   flutter run -d android
